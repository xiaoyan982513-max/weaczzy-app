# Weaczzy GitHub Pages

TikTok Shop管理工具 Weaczzy 的官方网站。

## 文件说明

- `index.html` - 主页/应用介绍
- `terms.html` - 服务条款
- `privacy.html` - 隐私政策

## GitHub Pages 设置步骤

1. 在GitHub创建新仓库，命名为 `weaczzy-app`
2. 上传这三个文件到仓库
3. 进入 Settings → Pages
4. Source 选择 "Deploy from a branch"
5. Branch 选择 "main" 和 "/ (root)"
6. 点击 Save
7. 等待几分钟，访问 `https://你的用户名.github.io/weaczzy-app/`

## TikTok开发者配置

配置完成后，在TikTok开发者后台填写以下URL：

- **Terms of Service URL**: `https://你的用户名.github.io/weaczzy-app/terms.html`
- **Privacy Policy URL**: `https://你的用户名.github.io/weaczzy-app/privacy.html`
- **Web/Desktop URL**: `https://你的用户名.github.io/weaczzy-app/`
